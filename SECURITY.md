# Security Policy

| Version | Supported |
|---------|-----------|

## Reporting a Vulnerability
Please do NOT report security vulnerabilities
through public GitHub issues.

Email: security@persistentresearch.in
Response time: Within 24 hours and 
Resolution target: Within 72 hours

We take all security reports seriously
and will acknowledge your contribution.
