# Contributing to Persistent Research

Thank you for your interest in contributing.

## Ways to Contribute
- Report bugs and issues
- Suggest improvements
- Submit pull requests
- Improve documentation
- Share research findings

## Pull Request Process
1. Fork the repository
2. Create a feature branch
3. Make your changes with clear commit messages
4. Submit a pull request with description
5. Wait for review - we aim to respond within 72 hours

## Research Contributions
For research contributions including
model improvements and evaluation results,
please email research@persistentresearch.in
before submitting a PR.

## Questions
Open an issue or email
research@persistentresearch.in
