# Persistent Research

**Frontier AI research company building open models from India for the world.**

We research, train, and deploy an ecosystem of frontier AI models - accessible on any device, online or offline, inside a sandboxed permission-free runtime.

---

### What We Build

| Product | Description |
|---|---|
| **Persistent Open** | Open-weight SLM series on HuggingFace |
| **Persistent Frontier** | Flagship frontier LLM |
| **Persistent Bench** | Open evaluation framework |
| **Persistent CLI** | Terminal interface for developers |
| **Persistent API** | Developer platform and SDK |

---

### Research Focus

- Continual learning and memory architecture
- Efficient frontier model training
- Multimodal unification
- Alignment and safety
- Offline-capable inference
